# JobStudyAgent 测试样例

## 1. 启动服务

```bash
uvicorn app.main:app --reload
```

Swagger 地址：

```text
http://127.0.0.1:8000/docs
```

---

## 2. 上传知识库

接口：

`POST /agent/upload-index`

建议参数：

- `index_name`: `agent-study-demo`
- `chunk_size`: `100`
- `chunk_overlap`: `20`
- `chunk_strategy`: `paragraph`
- `file`: `G:\study\codex-ai\doc\第三章-Agent学习总结.md`

预期结果：

- 返回 `index_name`
- 返回 `chunk_count`
- 返回 `embedding_model`

---

## 3. 查看知识库列表

接口：

`GET /agent/indexes`

预期结果：

- 返回 `agent-study-demo`
- 返回 `chunk_count`
- 返回 `chunk_strategy`

---

## 4. 单独测试知识库检索

接口：

`POST /agent/search`

请求体：

```json
{
  "index_name": "agent-study-demo",
  "question": "LangGraph 是什么",
  "top_k": 3,
  "score_threshold": 0.0,
  "source_name_filter": ""
}
```

预期结果：

- `selected_chunks` 不为空
- `scores` 不为空
- `source_names` 不为空

## 5. 测试普通聊天

接口：

`POST /agent/chat`

请求体：

```json
{
  "message": "请用通俗的话解释一下什么是 FastAPI",
  "session_id": "demo-chat",
  "index_name": "",
  "debug": true
}
```

预期结果：
